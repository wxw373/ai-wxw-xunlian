# AI知识问答助手 - 启动说明

## 🚀 如何启动

### 方式1：使用PowerShell脚本（推荐）

**右键点击以下文件，选择"使用PowerShell运行"：**
- `start_interactive.ps1` - 启动命令行交互模式
- `start_web.ps1` - 启动Web图形界面

**或者在PowerShell中运行：**
```powershell
.\start_interactive.ps1
# 或
.\start_web.ps1
```

### 方式2：使用BAT批处理文件

双击运行：
- `start_interactive.bat` - 启动命令行交互模式
- `start_web.bat` - 启动Web图形界面

### 方式3：直接命令行

在PowerShell或CMD中运行：

```bash
# 交互模式
.\venv\Scripts\python.exe main.py --mode interactive

# Web界面
.\venv\Scripts\python.exe -m streamlit run web_app.py

# 查看系统信息
.\venv\Scripts\python.exe main.py --mode info
```

## 📝 使用提示

### 交互模式
- 输入问题后按回车
- 输入 `exit` 退出
- 输入 `info` 查看系统信息

### Web界面
- 访问地址：http://localhost:8501
- 按 Ctrl+C 停止服务器

## ⚠️ 如果遇到问题

### PowerShell执行策略错误
运行以下命令允许脚本执行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 找不到虚拟环境
确保在项目根目录运行脚本，或检查venv目录是否存在。

### API密钥错误
检查 `.env` 文件中的API密钥是否正确设置。
