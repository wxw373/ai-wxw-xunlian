## 🚀 启动方式

### 方式1：PowerShell脚本（推荐）

在项目目录下，右键点击以下文件，选择"使用PowerShell运行"：

```
start.ps1
```

或者在PowerShell中运行：

```powershell
.\start.ps1
```

### 方式2：直接运行Python

```powershell
.\venv\Scripts\python.exe main.py --mode interactive
```

## 📝 使用流程

1. **启动系统**
   ```powershell
   .\start.ps1
   ```

2. **等待初始化**
   - 首次运行需要创建向量索引（约1-2分钟）
   - 后续启动会很快（几秒钟）

3. **开始提问**
   ```
   请输入问题: 什么是人工智能？
   ```

4. **查看答案**
   - 系统会显示答案
   - 显示参考来源
   - 显示相关文档片段

5. **继续提问或退出**
   - 继续提问：直接输入问题
   - 查看信息：输入 `info`
   - 退出系统：输入 `exit`

## 📂 添加自己的文档

### 步骤1：添加文档
将您的文档（PDF、TXT、MD、DOCX）放入：
```
data\documents\
```

### 步骤2：重建索引
```powershell
.\venv\Scripts\python.exe main.py --mode index --force
```

### 步骤3：重新启动
```powershell
.\start.ps1
```

## ⚠️ 注意事项

1. **首次运行**
   - 需要下载嵌入模型（约400MB）
   - 创建向量索引需要几分钟
   - 请确保网络连接正常

2. **API密钥**
   - DeepSeek API密钥已配置
   - 密钥位置：`.env` 文件
   - 请勿泄露您的API密钥

3. **文档格式**
   - 支持：PDF、TXT、Markdown、Word
   - 建议：单个文档不超过50MB
   - 编码：推荐UTF-8

4. **性能优化**
   - 文档越多，索引时间越长
   - 建议定期清理不需要的文档
   - 可调整 `config.py` 中的参数
