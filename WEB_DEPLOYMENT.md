# AI知识问答助手 - Web部署指南

## 🌐 Web应用部署

您的AI知识问答助手现在可以通过Web界面让其他人使用了！

## 🚀 快速启动

### 方式1：本地访问（推荐）

**双击运行：**
```
start_web_app.bat
```

**或在PowerShell中运行：**
```powershell
.\start_web_app.ps1
```

**访问地址：**
```
http://localhost:8501
```

### 方式2：局域网访问

如果您想让同一局域网内的其他设备访问，需要：

1. **查看本机IP地址**
   ```powershell
   ipconfig
   ```
   找到"IPv4 地址"，例如：`192.168.1.100`

2. **启动Web服务**
   ```powershell
   .\start_web_app.ps1
   ```

3. **其他设备访问**
   ```
   http://192.168.1.100:8501
   ```

### 方式3：公网访问（需要额外配置）

要让互联网上的用户访问，您需要：

#### 选项A：使用ngrok（推荐）

1. **安装ngrok**
   - 访问：https://ngrok.com/
   - 下载并安装ngrok

2. **启动Web服务**
   ```powershell
   .\start_web_app.ps1
   ```

3. **在另一个终端运行ngrok**
   ```powershell
   ngrok http 8501
   ```

4. **获取公网地址**
   ngrok会显示类似：
   ```
   Forwarding: https://xxxx-xxxx.ngrok.io -> http://localhost:8501
   ```

5. **分享链接**
   将 `https://xxxx-xxxx.ngrok.io` 分享给其他人

#### 选项B：使用云服务器

1. **购买云服务器**（阿里云、腾讯云等）
2. **上传代码到服务器**
3. **安装依赖**
4. **启动服务**
5. **配置防火墙开放8501端口**

## 🔧 高级配置

### 修改端口

编辑 `start_web_app.ps1`，修改端口号：
```powershell
--server.port 8502
```

### 限制访问IP

默认允许所有IP访问（`0.0.0.0`），如需限制：
```powershell
--server.address 127.0.0.1
```

### 启用密码保护

1. **创建配置文件** `.streamlit/config.toml`
   ```toml
   [server]
   enableCORS = false
   enableXsrfProtection = true
   
   [auth]
   enabled = true
   username = "admin"
   password = "your_password"
   ```

## 📱 移动端访问

## 📞 技术支持

### 常见问题

**Q: Web界面打不开？**
A: 检查防火墙是否开放8501端口

**Q: 其他设备无法访问？**
A: 确保在同一局域网，检查IP地址

**Q: 如何修改API密钥？**
A: 编辑 `.env` 文件，重启服务

**Q: 如何添加文档？**
A: 将文档放入 `data/documents/` 目录，重启服务
